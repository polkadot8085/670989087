Пентестинг (пенетрационное тестирование) сам по себе не является незаконным, если проводится с разрешения владельца системы или сети. Это важный аспект кибербезопасности, который помогает выявить уязвимости и улучшить защиту информационных систем.

Однако, если проводить пентестинг без разрешения, это будет считаться незаконным и может привести к серьезным правовым последствиям. Незаконное проникновение в чужие системы (даже в целях тестирования) нарушает законы о компьютерных преступлениях и может быть расценено как хакерская атака.

Законный пентестинг обычно проводится в рамках договоренностей и соглашений между заказчиком и пентестером, где четко оговорены границы тестирования, цели и ожидаемые результаты.

   ```python
   import jwt
   import datetime

   SECRET_KEY = 'your_secret_key'

   def generate_token(user_id):
       expiration = datetime.datetime.utcnow() + datetime.timedelta(hours=1)
       payload = {'user_id': user_id, 'exp': expiration}
       return jwt.encode(payload, SECRET_KEY, algorithm='HS256')
   ```

**1.1.2. Проверка токенов:**
   ```python
   def verify_token(token):
       try:
           payload = jwt.decode(token, SECRET_KEY, algorithms=['HS256'])
           return payload['user_id']
       except jwt.ExpiredSignatureError:
           return None
       except jwt.InvalidTokenError:
           return None
   ```
